What I do

Responsive Web Design

With the use of CSS transitions and Media Queries, flexible visuals, CSS Grid and Flex-box I create responsive web designs so that my websites maintain a consistent look and feel across multiple devices. Thinking and planning responsiveness is an integral part of my workflow.

Frameworks

A couple of my works are based on Vanilla JS and use jQuery, but now I primarily use React to build functional components for my websites and mobile apps. I'm currently improving my React and responsive web design skills. For back-end I prefer working with Laravel as it is open-source and follows the model-view-controller architectural pattern.

Skills

Version Control with Git is an integral part of my working methodology as well as testing and debugging. I have experience working with CSS preprocessors such as SASS. I also use RESTful APIs when creating new applications.
